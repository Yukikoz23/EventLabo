Event Labo

JavaScript のイベントを体感しよう

ボタンをクリックするとイベントが発火して
イベント名が表示されます。

Mouse Event

ボタンをクリックしてみよう

  • click

    クリックされた時

  • dblclick

    ダブルクリックされた時

  • contextmenu

    右クリックされた時

  • mousedown

    ボタンが押された時

  • mouseup

    ボタンが離された時

マウスポインタを要素に乗せてみよう

子要素と親要素の違いは・・・？

Parent element

Child Element

  • mouseover

    マウスポインタが要素に乗った時

  • mouseenter

    マウスポインタが要素に乗った時
    ( 親要素のみ発火 )

  • mouseout

    マウスポインタが要素から離れた時

  • mouseleave

    マウスポインタが要素から離れた時
    ( 親要素のみ発火 )

  • mousemove

    マウスポインタが要素の上を移動した時

入力したテキストを選択してみよう


  • select

    入力テキストが選択されている時

Touch Event

タッチしてみよう

  • touchstart

    指を置いた時

  • touchend

    指を離した時

  • touchmove

    指を置いたまま画面上を移動した時

Keyboard Event

テキストを入力してみよう

半角入力と全角入力の違いは・・・？

  • keydown

    キーボードが押された時

  • keypress

    キーボードが押された時
    ( 文字キーのみ/半角入力のみ発火 )

  • keyup

    キーボードから離れた時

Drag & Drop Event

フラスコを下の要素にドラッグ＆ドロップしてみよう

  • dragstart

    ドラッグが開始された時

  • drag

    ドラッグされている時

  • dragend

    ドラッグが終了した時

  • dragenter

    ドラッグ要素がドロップ要素に
    乗った時

  • dragover

    ドラッグ要素がドロップ要素に
    重なっている時

  • dragleave

    ドラッグ要素がドロップ要素から
    離れた時

  • drop

    ドロップされた時

CSS Transition Event

手にマウスオーバーしてみよう

  • transitionend

    CSS Transitionが終了した時

Value Change Event

Form Event

フォームに入力してボタンをクリックしてみよう

  • change

    <form>要素内の値が変更されて
    <form>要素からフォーカスが離れた時

  • input

    <input>や<textarea>要素の
    値が変更された時

  • submit

    送信ボタンが押された時

  • reset

    リセットボタンが押された時

Clipboard Event

テキストをコピー/切取り/ペーストしてみよう

  • cut

    選択範囲が
    切り取られた時

  • copy

    選択範囲が
    コピーされた時

  • paste

    クリップボードのアイテムが
    貼り付けられた時

Focus Event

テキスト入力欄にフォーカスを当ててみよう

  • focus

    要素がフォーカス
    されている時

  • blur

    要素がフォーカスを
    失っている時

View Event

ウインドウをリサイズ/スクロールしてみよう

  • resize

    ウインドウがリサイズされた時

  • scroll

    スクロールされた時

Media file Event

動画を再生/一時停止してみよう

  • play

    メディアファイルが
    再生された時

  • playing

    メディアファイルが再生中の時

  • pause

    メディアファイルの
    再生が一時停止の時

  • progress

    メディアファイルを読み込み中の時
